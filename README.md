# Vanilla Todo App

A simple Todo List application built using HTML, CSS, and Vanilla JavaScript.

## Preview

A simple web application to manage daily tasks. Users can add, delete, and manage their todo items.

## Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Display task list dynamically using JavaScript DOM manipulation

## Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)
